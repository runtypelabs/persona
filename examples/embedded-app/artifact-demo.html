Artifact sidebar

Two setups: an inline embed and a floating launcher (bottom-right). Both use features.artifacts. Inline toolbar talks to the embedded instance; launcher toolbar opens the bubble first, then runs the same actions. Globals: window.personaArtifactDemo and window.personaArtifactLauncherDemo.

  • Markdown — rendered note in the artifact pane.
  • Known component — ArtifactDemoPill is registered in the demo.
  • Unknown component — fallback inspector card in the pane.
