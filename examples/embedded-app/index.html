Quick Start

Choose your integration path

Start with Runtype in minutes, or wire Persona into your own SSE backend.

01

Install Persona

Add the widget package to your app.

02

Sign up & create an agent

Run the guided setup to create your account and product, then create an agent.

Then create an agent and note the returned ID:

03

Create a client token

Generate a browser-safe token for the agent you just created.

Replace <agent-id> with the ID from step 02, then paste the returned token into the config below.

04

Mount the widget

Import the stylesheet and initialize Persona with the client token.

import "@runtypelabs/persona/widget.css";
import { DEFAULT_WIDGET_CONFIG, initAgentWidget } from "@runtypelabs/persona";

initAgentWidget({
  target: "#chat",
  config: {
    ...DEFAULT_WIDGET_CONFIG,
    clientToken: "ct_test_...",
    launcher: {
      ...DEFAULT_WIDGET_CONFIG.launcher,
      enabled: false,
    },
  },
});

Need server-side flow control instead? Use @runtypelabs/persona-proxy and switch to a proxy-based Runtype setup later.

Bring your own backend: point Persona at any SSE endpoint and adapt the request or event format if needed.
01

Install Persona

Add the widget package to your app.

02

Mount the widget

Import the stylesheet, initialize Persona, and point it at your SSE endpoint.

import "@runtypelabs/persona/widget.css";
import { initAgentWidget } from "@runtypelabs/persona";

initAgentWidget({
  target: "#chat",
  config: { apiUrl: "https://your-api.com/chat" },
});

If your backend uses a different request or event shape, customize customFetch or parseSSEEvent in the widget config.
