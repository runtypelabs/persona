Dynamic Forms

AI-generated forms via component middleware. When your backend streams a JSON component directive (e.g., {"component": "DynamicForm", "props": {...}}), the widget renders an interactive form.

Preview

Render the same DynamicForm bubble the AI would emit: no LLM round-trip.

Notes

How it works

The Preview form button uses controller.injectComponentDirective(...) to render the form directly from host code: the same path the LLM uses, but instant. Useful for QA, design previews, debug toggles, and local tools.

Layout variants

Same fields, three formStyles presets. The form component is unchanged: only its visual tokens vary. Each tab shows the formStyles config that produced the form on the right.
