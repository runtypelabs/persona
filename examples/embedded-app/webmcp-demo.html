WebMCP — Page tools

This page registers two tools through navigator.modelContext.registerTool() (via @runtypelabs/webmcp-polyfill). Persona snapshots them on every dispatch and ships them as clientTools[]. When the agent calls one, the widget executes it on the page and posts the result back via /resume.

Registered tools

  • search_products — annotated readOnlyHint: true. Returns mock catalog hits.
  • add_to_cart — sensitive operation. Calls client.requestUserInteraction for a per-call confirm.
