🎨 Custom Components Demo

This page demonstrates custom components with the Persona SDK. Components can be streamed in as JSON with props that update incrementally.

How it works

When the AI responds with a JSON object containing a component field, the SDK will:

  • Look up the component in the registry
  • Render it with the provided props
  • Update the component as props stream in incrementally

Try asking: "Show me a product card" or "Display a chart"
