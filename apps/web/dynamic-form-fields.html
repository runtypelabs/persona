Notes

How it works

Each variant mounts a fresh widget and uses injectComponentDirective(...) to render the form on selection: no LLM round-trip. Switch to Code (top-right) and open the Scenario tab to see the directive props JSON.

Use these as

A copy-paste template when authoring prompts that tell the model what fields to emit, or a smoke test for theme tweaks.

Variants

Field types: text, email, tel, url, number, date, time, textarea
Layout widths: full vs half (paired half fields share a row)
Helpers + required: helper_text, red asterisk on required fields
