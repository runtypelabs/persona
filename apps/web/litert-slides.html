Deckmate Gemma 4 · on-device
This demo needs WebGPU, which this browser doesn't expose. Try the latest Chrome or Edge on a machine with a supported GPU.

The assistant is Gemma 4 running in your browser via LiteRT-LM over WebGPU — no server, no API key. It drives this deck through the page's WebMCP tools. Watch the on-device eval panel to judge tool-calling latency and throughput.
