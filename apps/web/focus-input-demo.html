Focus Input

Three ways to control chat input focus: autoFocusInput config, controller.focusInput(), and the persona:focusInput DOM event.

Focus controls

For launcher mode, open the launcher to see auto-focus. For inline mode, the change takes effect on next message.

How it works

The autoFocusInput config focuses the input after the panel opens (~200ms). focusInput() programmatically focuses and returns false if the launcher panel is closed. The window event accepts a target instanceId.
